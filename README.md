To use `gurudo.se`:

1. Add a `CNAME` file with content: `gurudo.se`
2. In your DNS settings, add these A records:
   ```
   185.199.108.153
   185.199.109.153
   185.199.110.153
   185.199.111.153
   ```
3. Wait for DNS propagation (up to 48 hours)
