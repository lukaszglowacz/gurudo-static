About the Project

Gurudo Geo is a professional-grade time tracking and workforce management system that leverages GPS technology to ensure employees are physically present at their designated workplace before they can clock in.

  • Employees can only start work when within 100 meters of the workplace
  • Real-time GPS location verification using Haversine formula
  • Comprehensive time tracking with automated daily and monthly summaries
  • Multi-workplace and team management dashboard
  • Subscription-based billing with per-seat model via Stripe
  • Automated email workflows and event tracking with Klaviyo
  • PDF reports generation for work sessions
  • Mobile-first responsive design (Swedish UI)
  • Secure authentication with JWT tokens

Getting Started

To deploy your own instance of Gurudo Geo:

  • Clone both backend and frontend repositories from GitHub
  • Set up PostgreSQL database and configure connection settings
  • Configure environment variables for Django, Stripe API keys, and Klaviyo integration
  • Install Python dependencies and run Django migrations
  • Install Node.js dependencies for the React frontend
  • Configure the frontend API baseURL to point to your Django backend
  • Run development servers or deploy to production hosting
  • Customize subscription tiers, pricing, and geolocation radius as needed

Detailed documentation, installation guides, and system requirements are available in each repository's README file.
